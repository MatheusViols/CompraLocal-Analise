Os três tipos de usuário(Cliente, Lojista e Entregador) devem ser capazes de realizar cadastro na plataforma.
Essa ação requer informações diferentes para cada tipo de usuário, sendo elas:
### Cliente

- #### Obrigatórios
	- Nome completo para identificação;
	- CPF para identificação;
	- Endereço para localização;
	- Estado para localização;
	- Cidade para localização;
	- Numero de telefone para contato;
	- Email pessoal para um login seguro;
	- Senha de no minimo 8 dígitos para um login seguro;

- #### Opcionais no cadastro, mas necessários para uso
	- Dados do cartão para compras usando esse meio de pagamento.
	

## Lojista

- #### Obrigatórios
	- Nome da loja para identificação;
	- CNPJ para identificação;
	- Endereço para localização.
	- Estado para localização;
	- Cidade para localização;
	- Email corporativo para um login seguro;
	- Senha de no minimo 8 dígitos para um login seguro;

## Entregador

- #### Obrigatórios
	- Nome completo para identificação;
	- CPF para identificação;
	- Endereço para localização;
	- Estado onde atua para localização;
	- Cidade onde atua para localização;
	- Email pessoal para um login seguro;
	- Senha de no minimo 8 dígitos para um login seguro;
