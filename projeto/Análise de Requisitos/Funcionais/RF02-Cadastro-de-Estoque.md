Lojas podem cadastrar seus produtos. Esse é um requisito necessário para as funcionalidades básicas de compra e venda.

O cadastro é feito individualmente para cada produto e guarda as seguintes informações:

#### Obrigatórias
- Categoria para organização e identificação;
- Nome do produto para identificação;
- Preço do produto para compra e venda;
- Quantidade em estoque para compra e venda;
